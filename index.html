Yelp Camp

This web application, created during Colt Steele's The Web Developer Bootcamp, is a replica of Yelp that focuses on campsites. Users can register an account, post reviews, edit them, rate campsites, and leave comments. The application allows for easy browsing of different campsites, providing users with a convenient platform to share and access information on their favorite outdoor destinations.

  • Node.js
  • Express
  • MongoDB
  • EJS
  • Javascript
  • Passport.js
